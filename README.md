# Option-Greeks-Pricer
Engineered an interactive Black-Scholes-Merton options pricer in Streamlit computing call/put prices and all five Greeks (Δ, Γ, Θ, ν, ρ) analytically via SciPy. Visualizes Greek sensitivity across the full spot price range — revealing non-linear risk behavior from deep OTM to deep ITM under dynamic market assumptions.
